# Bubble sort

**Functioning** - Bubble sort is repeatedly swapping adjacent element on condition.

Example:
First Pass:
( **5** **1** 4 2 8 ) –> ( **1** **5** 4 2 8 )
( 1 **5** **4** 2 8 ) –> ( 1 **4** **5** 2 8 )
( 1 4 **5** **2** 8 ) –> ( 1 4 **2** **5** 8 )
( 1 4 2 **5** **8** ) –> ( 1 4 2 **5** **8** )

Second Pass:
( **1** **4** 2 5 8 ) –> ( **1** **4** 2 5 8 )
( 1 **4** **2** 5 8 ) –> ( 1 **2** **4** 5 8 )
( 1 2 **4** **5** 8 ) –> ( 1 2 **4** **5** 8 )
( 1 2 4 **5** **8** ) –> ( 1 2 4 **5** **8** )

Third Pass:
( **1** **2** 4 5 8 ) –> ( **1** **2** 4 5 8 )
( 1 **2** **4** 5 8 ) –> ( 1 **2** **4** 5 8 )
( 1 2 **4** **5** 8 ) –> ( 1 2 **4** **5** 8 )
( 1 2 4 **5** **8** ) –> ( 1 2 4 **5** **8** )

**Time complexity** O(n \* n)
